### Function

- funções executam ações:
  - enviar um valor e pegar um valor de volta.
  - mesmas regras de nomes das variáveis:
    - não pode começar com números.
    - não pode ter espaços.
  - tudo dentro de uma função pertence a ela
  - funções podem ter parâmetros
    - parâmetro é igual a uma variável
        > function saudacao(nome) // o parâmetro aqui é nome
    - recebe um argumento

- return
    - posso retornar qualquer item, objeto, valor...
    - quando ele encontra o return, o interpretador entende que nada mais na função será executado
        - exemplo:
        function soma(x, y) {
        const resultado = x + y;
        return resultado;
        console.log(soma(2, 2)); // não será executado
    }

- tudo o que está dentro de uma função está protegido
    - exemplo:
    > function soma(x, y) {
    const resultado = x + y;
    return resultado;
    }
    // não consigo usar o que estiver na variável 'resultado' pq ela está dentro do escopo, então não posso dar um console.log(resultado); já que isso está 'protegido'.
    - posso acessar isso reutilizando:
    > const resultado = soma(2, 2);
    console.log(resultado);
